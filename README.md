# Simple Login Validation Form

A simple login validation form built using **HTML, CSS, and JavaScript**.  
This project validates user input and checks password strength based on specific rules.

---

## âœ… Features

- Validates empty username and password
- Prevents spaces in username and password
- Password must be at least **8 characters**
- Password must contain:
  - **At least one uppercase letter**
  - **At least one number**
  - **Must include @ or #**
- Displays error messages
- Reset button clears the fields
